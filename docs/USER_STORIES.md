## 🎨 选品设计师 (designer)

### 故事1：提交选品需求（Tech Pack 标准化）

> **作为**选品设计师，**我想**通过标准化的Tech Pack提交选品需求，**以便**AI Agent能精准理解我的设计意图，而非依赖模糊的文字描述。

**操作路径：**
1. 登录 → 选择「选品设计师」角色
2. 进入「AI 选品设计」页面
3. 选择行业（默认服装鞋包，支持6个行业切换）
4. 填写结构化字段：品类/季节/客群/版型/面料/工艺/尺码
5. 设置成本上限和生成数量
6. 点击「生成AI Prompt」→ 系统自动从结构化工单组装Prompt
7. 查看Prompt预览 → 确认后启动Agent协同执行

**验证点：** 切换行业时表单字段动态变化；所有字段收集完整；Prompt包含全部选中参数

### 故事2：查看Agent协同执行过程

> **作为**选品设计师，**我想**实时看到5个AI Agent的协同执行过程，**以便**了解每个环节在做什么、数据从哪里来。

**Agent执行流程：**
1. 🧠 总管Agent → 解析Tech Pack，拆解任务
2. 🔍 趋势Agent → MCP爬虫采集IG/Pinterest潮流，Milvus向量RAG检索
3. 📊 PLM数据Agent → MCP数据库查询热销款销量/退货/BOM成本
4. 🎨 设计Agent → Stable Diffusion生成效果图，CLIP拆解版型面料
5. 💰 供应链Agent → 面料+加工成本核算，MOQ拦截超预算方案

**验证点：** 5个步骤依次执行（动画展示）；每步有具体描述文字；执行后自动提交供应链审批

### 故事3：查看选品报告

> **作为**选品设计师，**我想**查看历史选品报告，**以便**了解之前生成的方案和审批状态。

**操作路径：**
1. 进入「选品报告」页面
2. 浏览报告列表（按时间倒序）
3. 点击「详情」查看完整方案 → 含款式卡片、成本明细、爆款率预测
4. 下载PDF版本用于会议汇报

### 故事4：管理选品任务

> **作为**选品设计师，**我想**查看所有选品任务的状态，**以便**跟进待审批/驳回的方案。

**操作路径：**
1. 进入「我的选品任务」页面
2. 查看任务列表：处理中/待审批/已通过/已驳回
3. 驳回的方案点击「查看」跳转到设计页重新编辑

### 故事5：浏览趋势数据

> **作为**选品设计师，**我想**查看MCP爬虫采集到的市场趋势数据，**以便**数据驱动地做选品决策。

**操作路径：**
1. 进入「趋势研究」页面
2. 查看各款式趋势指数（基于IG/Pinterest/TikTok数据）
3. 按品类筛选趋势 → 选择感兴趣的趋势作为选品参考

## 📦 供应链专员 (supply)

### 故事6：审批款式成本

> **作为**供应链专员，**我想**审批设计师提交的AI生成款式成本方案，**以便**确保面料/辅料/加工费在合理范围内。

**操作路径：**
1. 登录 → 选择「供应链专员」角色
2. 进入「成本审批」页面 → 看到3个待审批方案
3. 查看每个方案的成本明细：面料/辅料/加工/水洗/管理费分项列出
4. 成本在预算内 → 点「通过」→ Toast提示已通知设计师
5. 成本超预算 → 点「驳回」→ Toast提示设计师收到修改建议

**验证点：** 审批按钮有确认反馈；Toast通知弹出；审批后状态更新

### 故事7：管理物料BOM数据

> **作为**供应链专员，**我想**管理物料BOM成本基准表，**以便**AI成本校验使用最新的价格数据。

**操作路径：**
1. 进入「物料BOM管理」页面
2. 查看8种物料的完整信息：编码/名称/单价/供应商/数据源/更新频率
3. 核对ERP同步数据 vs 供应商报价 vs 1688实时价格
4. 查看MOQ限制说明：面料500米起订、辅料200条起订的溢价比例

**验证点：** 物料表包含编码、数据源、更新频率列；MOQ说明面板可读

### 故事8：管理工厂产能

> **作为**供应链专员，**我想**查看合作工厂的产能与交期数据，**以便**判断方案是否可以被工厂承接。

**操作路径：**
1. 进入「工厂管理」页面
2. 查看4家工厂：品类特长/月产能/MOQ/交期/利用率/状态
3. 产能利用率88%（东莞虎门）→ 黄色预警，避免新单派发

**验证点：** 工厂表包含产能利用率和状态列；异常状态有颜色标记

## 📊 运营复盘专员 (ops)

### 故事9：查看销售数据看板

> **作为**运营复盘专员，**我想**查看AI选品方案的实际销售表现，**以便**对比预测与实际的偏差。

**操作路径：**
1. 登录 → 选择「运营复盘专员」角色
2. 进入「销售数据看板」页面
3. 查看核心指标：TOP1周销量15,230件、AI命中率78%、退货率3.2%
4. 浏览款式销售对比表：每个款式的AI预测 vs 实际销量 vs 退货率

**验证点：** 统计卡片4列显示；表格包含命中/偏差标记

### 故事10：选品复盘

> **作为**运营，**我想**复盘待分析的AI选品方案，**以便**总结爆款特征、改进选品策略。

**操作路径：**
1. 进入「选品复盘」页面
2. 查看待复盘方案列表：每个方案的预测爆款率 → 实际销量 → 命中/偏差
3. 命中方案（高腰阔腿牛仔）→ 总结爆款特征
4. 偏差方案（毛边磨破牛仔）→ 分析滞销原因

**验证点：** 复盘列表显示预测vs实际对比；有生成复盘按钮

### 故事11：查看复盘报告

> **作为**运营，**我想**查看生成的复盘报告存档，**以便**用于团队分享和下季度选品策略优化。

**操作路径：**
1. 进入「复盘报告」页面
2. 浏览历史报告列表
3. 下载或分享复盘报告

## ⚙️ AI管理员 (admin)

### 故事12：编排Agent工作流

> **作为**AI管理员，**我想**通过可视化画布编排Agent工作流，**以便**适配不同业务场景的执行需求。

**操作路径：**
1. 登录 → 选择「AI管理员」角色
2. 进入「工作流编排」页面
3. 从左侧Agent组件库拖拽节点到画布
4. 点击节点查看配置：模型/温度/超时/重试次数
5. 加载预设模板：标准选品/快速打版/成本优先/爆款复盘
6. 连线配置Agent执行顺序

**验证点：** 节点可以添加/拖拽/删除；模板加载正常；SVG连线渲染

### 故事13：监控Agent运行状态

> **作为**AI管理员，**我想**实时监控5个Agent的运行状态，**以便**发现异常及时处理。

**操作路径：**
1. 进入「Agent监控」页面
2. 查看5个Agent：运行中/GPU排队/异常
3. 监控指标：今日调用次数/平均延迟/错误率

**验证点：** 表格包含完整的Agent监控数据

### 故事14：配置MCP工具

> **作为**AI管理员，**我想**管理4类MCP Server的连接配置，**以便**Agent能正常调用外部数据和模型。

**操作路径：**
1. 进入「MCP工具配置」页面
2. 查看4类MCP服务：数据库/多模态/爬虫/PLM同步
3. 检查连接状态：已连接/代理过期 → 编辑配置

### 故事15：管理用户权限

> **作为**AI管理员，**我想**管理系统的用户和角色分配，**以便**确保数据安全和功能隔离。

**操作路径：**
1. 进入「用户管理」页面
2. 查看4个用户及角色分配
3. 各角色权限范围一目了然
