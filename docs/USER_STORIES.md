## 🎨 选品设计师 (Designer)

> **核心职责：** 使用 AI 快速生成符合市场趋势的款式方案，审核结果后提交供应链审批

### 场景1：新建选品（核心流程）

**前置条件：** 已登录，已有历史销售数据和供应商报价

**操作步骤：**
1. 登录 → 工作台 → 点击「AI 选品设计」
2. 选择行业品类（服装鞋包/电子配件/家居用品/美妆个护/运动户外/食品饮料）
3. 系统自动展示该行业专属 Tech Pack 结构化字段
4. 依次填写：品类/季节/客群/版型/面料/特殊工艺/尺码范围/成本上限/生成数量
5. 点击「生成 AI Prompt · 启动 Agent」
6. 查看 AI 自动生成的结构化 Prompt
7. 5个 Agent 依次执行：总管拆解任务 → 趋势 RAG 检索 → PLM 数据查询 → 多模态生成 → 供应链成本校验
8. 审核生成的3个款式方案（爆款率/成本/风格标签）
9. 方案自动提交供应链专员审核成本

**预期结果：** 生成3个在预算内的款式方案，Prompt 可溯源到结构化字段

### 场景2：切换行业选品

**操作步骤：**
1. AI选品设计 → 切换行业为「电子配件」
2. 字段自动变为：品类/材质/接口/电池容量/防水等级/风格
3. 填写后生成电子配件类 Prompt

**预期结果：** 字段随行业切换自动变化，Prompt 正确拼接新字段

### 场景3：管理选品任务

**操作步骤：**
1. 登录 → 我的选品任务
2. 查看任务状态：⏳待审批 / ✅已通过 / 🔄设计中 / ❌已驳回
3. 驳回的任务可查看供应链的修改建议

**预期结果：** 任务状态实时更新，驳回任务显示拒绝原因

### 场景4：查看趋势

**操作步骤：**
1. 登录 → 趋势研究
2. 查看各品类趋势指数和热门标签
3. 数据来源：Instagram + Pinterest + TikTok

**预期结果：** 显示4个爆款趋势及趋势指数

### 场景5：查看报告

**操作步骤：**
1. 登录 → 选品报告
2. 查看历史选品方案列表
3. 下载 PDF 版报告

**预期结果：** 3个历史报告可查看和下载

## 📦 供应链专员 (Supply Chain)

> **核心职责：** 审核 AI 生成款式的成本可行性，管理物料 BOM 和工厂产能

### 场景1：成本审批（核心流程）

**操作步骤：**
1. 登录 → 工作台 → 点击「成本审批」
2. 查看3个待审批方案（含成本明细：面料/辅料/加工/水洗/管理费）
3. 审核每个方案：成本 ≤ 上限 → ✅ 通过；成本超上限 → ❌ 驳回
4. 操作后 Toast 通知设计师

**预期结果：** 审批后弹窗通知，设计师任务列表状态更新

### 场景2：管理物料 BOM

**操作步骤：**
1. 登录 → 物料 BOM 管理
2. 查看8种物料编码/成本/数据源/更新频率
3. 了解每个物料的成本采集渠道（ERP/供应商门户/1688/期货/合同价）
4. 查看 MOQ 限制和超量溢价规则

**预期结果：** 物料表显示完整编码体系、数据源和更新频率

### 场景3：管理工厂产能

**操作步骤：**
1. 登录 → 工厂管理
2. 查看4家合作工厂的月产能/MOQ/交期/产能利用率

**预期结果：** 工厂表显示真实业务参数

## 📊 运营复盘专员 (Operations)

> **核心职责：** 监控 AI 选品方案的实际销售表现，分析爆款/滞销原因

### 场景1：查看销售数据看板

**操作步骤：**
1. 登录 → 工作台 → 点击「销售数据看板」
2. 查看核心指标：TOP1周销量 / AI命中率 / 退货率 / 在售AI款式数
3. 查看款式销售对比表（AI预测 vs 实际销量）

**预期结果：** 3个款式显示预测-实际对比，标记✅命中或⚠️偏差

### 场景2：选品复盘

**操作步骤：**
1. 登录 → 选品复盘
2. 查看待复盘方案（预测爆款率 vs 实际销量）
3. 点击「生成复盘」生成分析报告

**预期结果：** 显示每个方案的预测 vs 实际，可生成复盘报告

## ⚙️ AI 管理员 (Admin)

> **核心职责：** 编排 Agent 工作流，监控系统状态，配置 MCP 服务和用户权限

### 场景1：编排工作流（核心流程）

**操作步骤：**
1. 登录 → 工作流编排
2. 选择预设模板：标准选品流程 / 快速打版流程 / 成本优先流程 / 爆款复盘流程
3. Agent 自动排列到画布上
4. 拖拽节点调整位置，SVG 自动连线
5. 点击节点查看配置：模型/Temperature/超时/重试

**预期结果：** 画布上显示 Agent 节点 + SVG 虚线连线

### 场景2：监控 Agent

**操作步骤：**
1. 登录 → Agent 监控
2. 查看5个 Agent 实时状态/今日调用/延迟/错误率

**预期结果：** Agent 表格显示运行状态和性能指标

### 场景3：配置 MCP 服务

**操作步骤：**
1. 登录 → MCP 工具配置
2. 查看4个 MCP Server 连接状态
3. 编辑服务器地址

**预期结果：** 数据库/多模态/爬虫/PLM同步 四类 MCP 状态可查看

### 场景4：用户权限管理

**操作步骤：**
1. 登录 → 用户管理
2. 查看4个用户及其角色权限范围
3. 设计师：选品/趋势/报告
4. 供应链：审批/物料/工厂
5. 运营：数据/复盘/报告
6. 管理员：全部权限

**预期结果：** 不同角色拥有不同的菜单和功能权限
